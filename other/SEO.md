# SEO 优化
----

**FLASH JS 语言不利于网络爬虫爬去**

通用方法：
+ 网站标题、网站关键字、描述
+ 网站内容优化
+ Robot.txt文件
+ 网站地图
+ 增加外链

前段方法：
+ 网站结构布局优化
+ 网页代码优化

布局优化
+ 扁平化结构
  - 控制首页链接数量（100个以内，关键字导航，内部导航）
  - 扁平化的目录层次（跳转三次到达任意页面，主页-栏目-目标）
  - 导航SEO优化（头部或者底部）
  - 面包屑导航 （方便了解位置，容易了解网站的组织形式）
  - 加载速度（页面小于100K）
  
细节
+ 分页
  - 首页-上页-下页-尾页
  - 首页 1 2 3 4 5 下页
  - 首页 1 2 3 4 5 下拉选择
  
网页代码优化
+ title 标题
+ `<meta keywords>`
+ `<meta description>`
+ 语意化代码
 - H1-h6
 - UL OL
 - DL 
 - em strong
 - a 标签 title rel='nofollow'

+ 原则
 - 重要的HTML代码放在前面
 - 重要内容不要用JS输出
 - 尽少使用iframe框架
 - 谨慎使用display:none 用z-index代替
 - 不断精简页面
